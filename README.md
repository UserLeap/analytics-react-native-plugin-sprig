# @segment/analytics-react-native-plugin-sprig

`DestinationPlugin` for [Sprig](https://www.sprig.com). This is a segment analytics plugin that wraps [`react-native-userleap`](https://www.npmjs.com/package/react-native-userleap).

Thank you for choosing Sprig. 

Please refer to our [Segment React Native SDK documentation](https://docs.sprig.com/docs/segment-react-native) to get started.

To report an issue with the iOS SDK, please contact your Sprig Customer Success representative.
